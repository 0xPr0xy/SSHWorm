IPAddress object pickling - IPv4.

{{{

>>> ip = IPAddress(3221225985)
>>> ip
IPAddress('192.0.2.1')

>>> buf = pickle.dumps(ip)

>>> ip2 = pickle.loads(buf)

>>> ip2 == ip
True

>>> id(ip2) != id(ip)
True

>>> ip2.value == 3221225985
True

>>> ip2.version
4

>>> del ip, buf, ip2

}}}
